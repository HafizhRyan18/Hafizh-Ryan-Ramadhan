Salah satu mata kuliah yang termasuk dalam KBK Sistem Informasi Geografis (SIG) adalah SIG Berbasis Internet. Pada mata kuliah ini mahasiswa mempelajari pengaplikasian SIG dengan berbasis pada platform internet. Mahasiswa mempelajari konsep kerja internet, proses request dan respond oleh user dan client, serta pembangunan website itu sendiri.

Memahami HTML dan CSS merupakan langkah selanjutnya bagi mahasiswa yang membuat halaman website. Kali ini, salah satu tugas adalah membangun tampilan halaman web sederhana ini. Selanjutnya halaman sederhana ini dikembangkan kembali dengan mengaplikasikan penggunaan Bootstrap.

Peta Statis

Peta statis adalah gambar peta tetap yang dibuat sekali dan tidak dapat dimanipulasi oleh pengguna. Peta ini disajikan sebagai file digital (seperti PNG, JPG, PDF) atau dalam bentuk cetakan fisik di atas kertas. Tidak ada fungsi interaksi seperti zoom, pan, atau pengubahan tampilan secara langsung oleh pengguna. Informasi yang ditampilkan pada peta statis bersifat tetap dan hanya dapat diperbarui dengan membuat peta baru.

Karakteristik:
  • Bersifat non-interaktif
  • Tampilan dan skala sudah ditentukan
  • Tidak menampilkan data terkini secara otomatis
  • Mudah digunakan dalam berbagai media

Contoh: peta lokasi di undangan, peta administrasi di buku atlas, peta geologi cetak.

Keunggulan:
  • Mudah dibuat dan didistribusikan
  • Tidak memerlukan internet
  • Cocok untuk dokumentasi tetap
Keterbatasan:
  • Tidak real-time
  • Tidak bisa diubah
  • Kurang fleksibel untuk analisis
Peta Dinamis

Peta dinamis adalah peta digital interaktif yang dihasilkan oleh server atau aplikasi web setiap kali pengguna melakukan aksi, seperti memperbesar (zoom), memindah (pan), atau memilih fitur tertentu. Data pada peta dinamis dapat diperbarui secara real-time, sehingga peta selalu mencerminkan kondisi atau informasi terkini sesuai permintaan pengguna.

Karakteristik:
  • Interaktif: zoom, pan, klik fitur
  • Data bisa berubah otomatis
  • Dapat menampilkan berbagai layer & fitur tambahan
  • Diakses via web/mobile/software
Keunggulan:
  • Real-time & eksploratif
  • Sangat fleksibel untuk monitoring & perencanaan
Keterbatasan:
  • Butuh koneksi internet
  • Lebih kompleks & berat sumber daya

Peta Interaktif

Peta interaktif adalah peta digital yang memungkinkan pengguna untuk berinteraksi langsung dengan data geografis yang ditampilkan, seperti menampilkan informasi, menambahkan data, hingga merencanakan rute perjalanan.
